12. `delta.png` - Delta logo

## Image Specifications

- **Format**: PNG (with transparent background preferred)
- **Recommended Size**: 300x150px (or similar aspect ratio)
- **Background**: Transparent or white
- **Quality**: High resolution for clarity

## Notes

- Logo files should have a transparent background for best appearance in both light and dark modes
- If the actual brand logo has a dark background, consider using a version with transparent background
- The system will auto-resize images to fit the container (120x60px display size)
